
# Install
- sudo apt-get install espeak
- http://espeak.sourceforge.net/commands.html

# Usage
#### get voices:
```espeak --voices=en```

#### example
```espeak -v en-polish "text"```

#### set voice speed: 
```espeak -s 175```

### Nice voice
```espeak -v en-scottish "Hello mister X, h
ow are you doing?"```
